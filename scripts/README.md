# Scripts

Utility scripts organized by purpose.

## Directories

### `setup/`
One-time setup and initialization scripts:
- `init_db.py` - Initialize database schema
- `init_local.py` - Local development setup

### `exploration/`
Ad-hoc analysis and data exploration scripts:
- `explore_data.py` - General data exploration
- `rigorous_analysis.py` - Statistical validation
- `deep_dive.py` - Deep analysis sessions
- `stability_analysis.py` - Data stability analysis

### `maintenance/`
Ongoing maintenance and operations:
- `compute_baselines.py` - Recompute personal baselines
